Catmandu exporter example
=========================

Step 1: Install the required libraries

     $ cpanm Catmandu Catmandu::BibTeX Catmandu::RIS Catmandu::XLS

Step 2: Execute the Perl scripts

     $ perl test.pl
     $ perl bibtex.pl
     $ perl csv.pl
     $ perl ris.pl
     $ perl xls.pl > test.xls
